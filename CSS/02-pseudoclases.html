Operadores

Para ello, aplicamos el operador ">" que indica que solo se aplicará al primer nivel dentro de la jerarquía de elementos. Pero, además de poder estilar hijos, también podríamos estilar hermanos. Para eso existe el operador "~", que aplica estilos a TODOS los hermanos que sigan después de un elemento "X".
